0. Introduction

บทเรียนนี้จะกล่าวถึงการวิเคราะห์ข้อมูลเบื้องต้น โดยใช้ภาษา R และแพ็กเกจต่าง ๆ ที่เกี่ยวข้อง เนื้อหาหลักประกอบด้วย

  • การเปรียบเทียบค่าเฉลี่ยระหว่างกลุ่มตัวอย่าง

  • การวิเคราะห์สหสัมพันธ์และการวิเคราะห์การถดถอย

ชุดข้อมูลสำหรับใช้เป็นตัวอย่าง

  • TeacherSalaryData

  • อีกชุดข้อมูลเป็นผลการตอบข้อสอบอัตนัยจากรายวิชาสถิติของนิสิตชั้นปีที่ 1 (แจกในห้องเรียน)

1. การเปรียบเทียบค่าเฉลี่ยระหว่างกลุ่มตัวอย่าง

  • การเปรียบเทียบว่ากลุ่มต่าง ๆ มีคุณลักษณะที่สนใจ (ตัวแปรตาม) แตกต่างกันอย่างไร และความแตกต่างดังกล่าวมีนัยสำคัญหรือไม่

  • การเปรียบเทียบคุณลักษณะดังกล่าวสามารถทำได้หลายมิติ มิติหลักที่นักสถิติมักใช้คือ ค่าเฉลี่ย ซึ่งเป็นแนวโน้มสู่ส่วนกลางของการแจกแจงของตัวแปร

  • อีกนัยนึงการเปรียบเทียบค่าเฉลี่ยสามารถมองเป็นการวิเคราะห์ความสัมพันธ์ระหว่างตัวแปรเชิงปริมาณกับตัวแปรจัดประเภทก็ได้เช่นเดียวกัน ขึ้นอยู่กับว่าผู้วิเคราะห์จะใช้สารสนเทศในลักษณะใด

การวิเคราะห์เพื่อเปรียบเทียบค่าเฉลี่ยระหว่างกลุ่มตัวอย่างอาจจำแนกได้เป็น 2 ลักษณะได้แก่

  1. การวิเคราะห์ด้วยทัศนภาพข้อมูล (data visualization) ร่วมกับสถิติบรรยาย ซึ่งมีหลายตัว/หลาย combination ที่สามารถนำมาใช้ได้

  • summary table

  • boxplot/violin/jitter plot

  • density plot

  • histogram/pyramid chart

  • cumulative distribution function plot (CDF plot)

  • bar chart/ bar chart + errorbar (ใช้ summary stat มา plot)

  1. การใช้สถิติวิเคราะห์ (statistical analysis) เช่น t-test, ANOVA หรือ Linear Model

## 3) รันกับคอลัมน์ answer และ “บังคับ schema” ตอน chat()
##    หมายเหตุ: ตั้ง .temperature = 0 ให้เสถียร
label_llm <- function(data, text_col = answer, model = "gpt-5-nano"){

text_sym <- rlang::enquo(text_col)
txt_vec <- data |> pull(!!text_sym) |> as.character()

message <- tibble(
    role = c("system","user"),
    content = c(system_prompt, glue::glue(user_prompt, TEXT = txt_vec))
)

out <- message |> 
    df_llm_message() |> 
    chat(openai(),
    .model = model,
    .json_schema = mainidea_schema) |> 
    get_reply_data()

return(out)
}

1.2 การใช้สถิติวิเคราะห์

เราสามารถจำแนกสถิติวิเคราะห์สำหรับการเปรียบเทียบค่าเฉลี่ยได้หลายโมเดล แต่จริง ๆ ทั้งหมดอยู่ภายในโมเดลเดียวกับคือ General Linear Model (GLM)

Independent-sample t-test

Independent two-sample t-test หรือบางครั้งเรียกว่า “Student’s t-test” ใช้เพื่อเปรียบเทียบค่าเฉลี่ยของตัวแปรต่อเนื่องระหว่างสองกลุ่มอิสระกัน เพื่อดูว่าค่าเฉลี่ยของทั้งสองกลุ่มมีความแตกต่างกันอย่างมีนัยสำคัญทางสถิติหรือไม่

\[ H_0: \mu_1 = \mu_2 \]

t.test() เป็นฟังก์ชันใน R ที่ใช้สำหรับการทดสอบ t-test ซึ่งสามารถใช้ได้ทั้งแบบ independent two-sample t-test และ paired t-test ขึ้นอยู่กับการกำหนดพารามิเตอร์

  • เราสามารถระบุการเปรียบเทียบได้สองลักษณะ คือ การใช้สูตร (formula interface) หรือการใช้เวกเตอร์ของข้อมูล (vector interface) ดังนี้ t.test(y ~ group, data = your_data) หรือ t.test(x = group1, y = group2) ทั้งนี้ขึ้นอยู่กับลักษณะของข้อมูลที่มี

  • var.equal เป็นพารามิเตอร์ที่ใช้ระบุว่าความแปรปรวนของสองกลุ่มที่เปรียบเทียบกันนั้นเท่ากันหรือไม่ (homogeneity of variance)

    • หากตั้งค่า var.equal = TRUE จะใช้ Student’s t-test ซึ่งสมมติฐานคือความแปรปรวนของทั้งสองกลุ่มเท่ากัน

    • หากตั้งค่า var.equal = FALSE จะใช้ Welch’s t-test ซึ่งไม่สมมติฐานว่า ความแปรปรวนของทั้งสองกลุ่มต้องเท่ากัน

  • alternative เป็นพารามิเตอร์ที่ใช้ระบุทิศทางของการทดสอบสมมติฐาน (hypothesis test) โดยมีตัวเลือกหลัก ๆ ดังนี้

    • "two.sided": ทดสอบว่า ค่าเฉลี่ยของสองกลุ่มไม่เท่ากัน (ค่าเริ่มต้น)

    • "less": ทดสอบว่า ค่าเฉลี่ยของกลุ่มแรกน้อยกว่ากลุ่มที่สอง

    • "greater": ทดสอบว่า ค่าเฉลี่ยของกลุ่มแรกมากกว่ากลุ่มที่สอง

ผลการทดสอบนัยสำคัญที่ได้เป็นอย่างไร?

ข้อตกลงเบื้องต้นของ independent two-sample t-test

  1. ตัวแปรตาม (dependent variable) เป็นตัวแปรเชิงปริมาณ (continuous variable)

  2. ตัวแปรอิสระ (independent variable) เป็นตัวแปรจัดประเภท (categorical variable) ที่มีสองกลุ่ม (two groups)

  3. กลุ่มตัวอย่างทั้งสองกลุ่มเป็นอิสระต่อกัน (independent samples)

  4. การแจกแจงของตัวแปรตามในแต่ละกลุ่มมีลักษณะใกล้เคียงกับการแจกแจงปกติ (normality assumption)

  5. ความแปรปรวนของตัวแปรตามในแต่ละกลุ่มมีความเท่ากัน (homogeneity of variance assumption) หรือจะไม่เท่าก็ได้ถ้าใช้ Welch’s t-test

  6. ข้อมูลไม่มีค่าผิดปกติ (outliers) ที่มีผลกระทบอย่างมากต่อผลการวิเคราะห์
